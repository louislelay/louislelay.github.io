I'm Louis Le Lay, a robotics engineer working on humanoids to make them agile through RL & simulation.

I’m an active contributor to Isaac Lab, and have recently helped to the creation of mjlab [tweet], a project to develop an Isaac Lab-style API on top of MuJoCo Warp.

My current work at PAL Robotics focuses on developing a RL-based locomotion for PAL's latest humanoid robot KANGAROO, using Isaac Lab.

Software

  • • mjlab (550+ stars)

    • Project led by Kevin Zakka. Combines Isaac Lab's proven API with best-in-class MuJoCo physics to provide lightweight, modular abstractions for RL robotics research and sim-to-real deployment.

  • • Kinova Gen3 RL & Sim2Real Toolkit (140+ stars)

    • Modular framework for training RL agents on the Kinova Gen3 robot using Isaac Lab, and deploying trained models either in Isaac Sim, or on the real robot via ROS2.

  • • SO-ARM RL: Manipulation tasks (100+ stars)

    • Implementation of RL-based manipulation tasks for the SO‑ARM100 and SO‑ARM101 robots using Isaac Lab.
